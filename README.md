# Weather App

This is a simple weather application built using HTML, CSS, and JavaScript.

The app allows users to enter a city name and see the current weather information like temperature and weather condition.

## Features
- Search weather by city
- Shows temperature
- Shows weather condition and icon

## Technologies Used
- HTML
- CSS
- JavaScript
- WeatherAPI
